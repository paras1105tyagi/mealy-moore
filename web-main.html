The Moore Machine

In a Moore machine, the output depends only on the current state. Think of a lighthouse: if it's ON, it shines. It doesn't matter how it got turned on.

Mathematical Tuple

Output Function λ: Q → Δ

Maps States (Q) directly to Output (Delta)

  • Output placement: Inside the state circle.
  • Analogy: A lighthouse shining.
  • Timing: Synchronous (safer).

The Mealy Machine

In a Mealy machine, the output depends on both the current state AND the input. It reacts during the transition itself.

Mathematical Tuple

Output Function λ: Q × Σ → Δ

Maps (State + Input) to Output

  • Output placement: On the arrow (edge).
  • Analogy: A piano key (Sound = State "Ready" + Input "Press").
  • Efficiency: Often uses fewer states.
